# EvidenceCompare AI — API Specification

**Phase:** 0 (Design) · **Base path:** `/api/v1` · **Last updated:** 2026-07-03

REST + Server-Sent Events. JSON everywhere. JWT bearer auth. OpenAPI is generated by
FastAPI at `/api/v1/openapi.json` and is the source of truth for shared types.

---

## Auth

### `POST /auth/register`
```json
// req
{ "email": "user@example.com", "password": "••••••••" }
// 201
{ "id": "uuid", "email": "user@example.com" }
```

### `POST /auth/login`
```json
// req
{ "email": "user@example.com", "password": "••••••••" }
// 200
{ "access_token": "jwt", "refresh_token": "jwt", "token_type": "bearer" }
```

### `POST /auth/refresh`
```json
{ "refresh_token": "jwt" } // → { "access_token": "jwt" }
```

### `GET /auth/me` → current user profile.

## Molecules

### `GET /molecules/search?q=telmi`
Normalized drug lookup (RxNorm/ATC), used by the input autocomplete.
```json
// 200
{ "results": [ { "id": "uuid", "name": "Telmisartan", "atc_code": "C09CA07", "synonyms": ["Micardis"] } ] }
```

## Reports

### `POST /reports`
Create and enqueue a comparison report.
```json
// req
{
  "molecule_a": "Telmisartan",
  "molecule_b": "Valsartan",
  "topic": "Cardioprotection",
  "options": { "population": "adults with CKD", "date_from": 2010, "sources": ["pubmed","ctgov","esc"] }
}
// 202
{ "report_id": "uuid", "status": "queued" }
```

### `GET /reports/{id}`
Full report (sections, comparison rows, citations) once available.
```json
// 200 (abbreviated)
{
  "id": "uuid",
  "status": "complete",
  "inputs": { "molecule_a": "Telmisartan", "molecule_b": "Valsartan", "topic": "Cardioprotection" },
  "sections": [
    { "section_key": "executive_summary", "title": "Executive Summary", "confidence": "moderate", "content": { "markdown": "...", "claims": [ { "text": "...", "citation_ids": ["uuid"] } ] } }
  ],
  "comparison": [
    { "attribute": "Reduction in CV events", "value_a": {...}, "value_b": {...}, "confidence": "high", "citation_ids": ["uuid"] }
  ],
  "citations": [
    { "id": "uuid", "doi": "10.1056/...", "pmid": "12345678", "title": "...", "verified": true }
  ],
  "cost_usd": 0.42
}
```

### `GET /reports/{id}/stream`  (SSE)
Live progress during generation. Events:
```
event: status        data: {"status":"running"}
event: agent         data: {"agent":"search","state":"done","docs":42}
event: section       data: {"section_key":"mechanism_of_action","confidence":"high"}
event: citation      data: {"citation_id":"uuid","verified":true}
event: complete      data: {"report_id":"uuid","cost_usd":0.42}
event: error         data: {"message":"..."}
```

### `GET /reports` → list current user's saved reports (paginated).
### `DELETE /reports/{id}` → delete a saved report.
### `POST /reports/{id}/share` → returns a share token/URL (read-only).

## Exports

### `POST /reports/{id}/exports`
```json
// req
{ "format": "pdf" }        // pdf | pptx | xlsx | markdown
// 202 → { "export_id": "uuid", "status": "processing" }
```
### `GET /exports/{id}` → `{ "status": "ready", "url": "https://.../report.pdf" }`

## Conventions

- **Auth:** `Authorization: Bearer <access_token>` on all non-auth routes.
- **Errors:** consistent envelope
  ```json
  { "error": { "type": "validation_error", "message": "topic is required", "request_id": "req_..." } }
  ```
- **Status codes:** 200/201/202 success; 400 validation; 401 auth; 403 forbidden;
  404 not found; 409 conflict; 429 rate limited (with `Retry-After`); 5xx server.
- **Rate limiting:** per-user token bucket; report creation is the most restricted route.
- **Pagination:** `?limit=&cursor=` with `{ "data": [...], "next_cursor": "..." }`.
- **Idempotency:** `POST /reports` accepts `Idempotency-Key`; identical
  `(A,B,topic,options)` may return a cached completed report.
- **Insufficient evidence:** sections/rows return `confidence: "very_low"` with an
  explicit `"insufficient_evidence": true` flag rather than fabricated content.
